\Abbreviations %% Список обозначений и сокращений в тексте
\begin{description}
    \item[ПО] программное обеспечение.
    \item[FOV] область видимости.
    \item[FOS] область резкости.
    \item[DOF] глубина резкости.
    \item[GPU] графический процессор.
    \item[WMS] Warehouse Management System — складская информационная система.
    \item[GA]  Genetic Algorithm — генетический алгоритм.
    \item[RL]  Reinforcement Learning — обучение с подкреплением.
    \item[MIP] Mixed-Integer Programming — смешанные целочисленные модели.
    \item[CP]  Constraint Programming — программирование ограничений.
    \item[GAPS] Greedy Algorithm for Pallet Stacking — жадный алгоритм укладки на паллету.
    \item[PBLS] Platform-Based Layered Strategy — стратегия послойной укладки на основе «платформ».
    \item[HAPS] Hybrid Algorithm for Pallet Stacking — гибридный алгоритм укладки на паллету.
    \item[LAPS] Layer-based Adaptive Packing Strategy — адаптивная послойная стратегия упаковки.
    \item[MAPS] Multi-Address Pallet Stacking — модуль адресного перераспределения коробок между паллетами.
    \item[3D-BPP] Three-Dimensional Bin Packing Problem — задача трехмерной упаковки в контейнеры.
    \item[FFD] First Fit Decreasing — алгоритм «Первый подходящий с убыванием», при котором объекты сначала сортируются от большего к меньшему.
    \item[PA] Packing Algorithm — Алгоритм упаковки.
    \item[VU] Volume Utilization — Коэффициент использования объема.
    \item[BL] Bottom-Left — эвристика «Нижний-Левый», стратегия размещения объекта в самую нижнюю и самую левую доступную позицию.
\end{description}
